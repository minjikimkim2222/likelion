Remember me '인증'

1. 세션이 만료되고, 웹브라우저가 종료된 후에도 어플리케이션이 계속 사용자를 기억하는 기능

2.  -> Remember-me 쿠키를 구워서 client에게 보내줌.
    -> Remember-me 쿠키를 확인해서, 토큰 기반으로 인증함.
    -> 토근이 검증이 되면, 해당 사용자로 로그인됨 !

    -> 실행결과, rememberme 체크하면, 쿠키값으로 remember-me가 만들어진 것 확인가능!
        ("세션이 만료되더라도", 사용자가 request로 가지고 들어온, remember-me 쿠키를 기반으로, '인증'을 시켜준다 !! )
